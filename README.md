# 208dowels_2018

USAGE

    ./208dowels O0 O1 O2 O3 O4 O5 O6 O7 O8

    Oi   size of the observed class
    
EXEMPLE

    ./208dowels 6 4 10 18 20 19 11 5 7
    
    ./208dowels 6 4 10 8 20 19 11 5 17
    
    ./208dowels 4 5 13 19 20 16 12 7 4
